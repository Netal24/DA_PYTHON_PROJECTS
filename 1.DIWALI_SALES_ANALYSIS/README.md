# DIWALI_SALES_DATA_ANALYSIS_PYTHON_1

## Project Overview

This project analyzes Diwali sales data to understand customer purchasing behavior during the festive season. The goal is to explore patterns in customer demographics, purchasing trends, and product category performance to generate useful business insights.

## Dataset

The dataset contains transactional sales records during the Diwali festival. It includes customer demographic details and purchase information.

Main columns in the dataset:

* User_ID
* Gender
* Age Group
* Marital Status
* State
* Product Category
* Amount (Purchase Amount)

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insight Generation

## Analysis Performed

* Customer distribution by gender
* Age group analysis
* State-wise sales analysis
* Total sales comparison by gender
* Product category sales performance

## Visualizations

The project includes several visualizations to better understand the data:

* Gender distribution of customers
* Age group distribution by gender
* Top states by total sales
* Sales comparison by gender
* Sales by product category

## Key Insights

* Female customers contribute more to total sales compared to male customers.
* The age group 26–35 has the highest number of buyers.
* Certain states generate significantly higher sales during the festival.
* Some product categories show higher demand during the Diwali season.
